------------------------------------------------------------------------------
  NOTES FOR WAVE POOL USAGE
------------------------------------------------------------------------------
  - ALL roles above under HOSTILE are Template_Intelligent or Template_Predator
    and will immediately aggro player in range. Safe for wave pools.
  - Avoid ChangeRole=true mobs (Goblin_Scavenger) — starts as passive hermit,
    transforms to hostile only after being attacked.
  - Dungeon_ prefixed variants are valid but designed for patrol path spawners;
    they will still aggro as regular combat NPCs when spawned via spawnNPC().
  - MaxHealth values are BASE values; the leveling system scales them further.
  - Wander variants (e.g. Skeleton_Burnt_Soldier_Wander) work but start in
    patrol mode; stationary versions are more predictable for waves.
  - Boss pool should use roles with HP >= 100 for impact; recommended:
    Goblin_Ogre, Goblin_Duke, Trork_Chieftain, Scarak_Broodmother,
    Zombie_Aberrant, Zombie_Aberrant_Big, Outlander_Marauder, Werewolf,
    Skeleton_Burnt_Praetorian, Crocodile.
